# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.2.x   | ✅        |
| < 0.2.2 | ❌        |

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not** open a public GitHub issue.

Please report it using [GitHub Security Advisories](https://github.com/fawadss1/scrapy-stealth/security/advisories/new).

Once the issue is confirmed and fixed, a patched release will be published.
